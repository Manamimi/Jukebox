Jukebox Commands

Show Help

Show help information.

help

Play Song (YouTube) (Guild Only)

Adds a song/playlist to the playback queue.

play <youtube_url | string>

Join Voice Channel (Guild Only)

Adds Jukebox to your current voice channel.

join

Skip Song (Guild Only)

Skips the current song and plays the next song in the queue.

skip

Remove Song (Guild Only)

Removes a song from the queue.

remove <songs | variadic (<queue_position | number>)>

Change Volume (Guild Only)

Changes the current volume of the bot. Volume must be 0-100 inclusive.

volume <volume | number>

Show Queue (Guild Only)

Shows the current queue of songs to be played.

queue

Show Now Playing (Guild Only)

Shows the currently playing song.

now

Search YouTube (Guild Only)

Searches YouTube.

search <query | variadic (<search_term | string>)>

Shuffle (Guild Only)

Shuffle the current playlist.

shuffle

Leave Channel (Guild Only)

Leave the current voice channel and delete the current queue.

leave

Show Information

Links to http://shiroi.moe (this page).

info

Advanced Commands

Remove (Advanced) (Guild Only)

There is an advanced remove command available for those who want it.

remove <remove <filters | variadic (<filter_token | string>)>

The following variables are available for use in filters:

  • title Song title
  • requester_id The ID of the user who requested the song
  • requester The user who requested the song (this should be a mention)
  • length The length of the song in seconds
  • number The number of the song in the queue
  • me Magic variable that represents you

The following operators are available for use in filters:

Operator Description
x contains y Returns true if x contains y (case-insensitive).
x is y Returns true if x equals y.
x is not y Returns true if x does not equal y.
x > y Returns true if x > y.
x >= y Returns true if x >= y.
x < y Returns true if x < y.
x <= y Returns true if x <= y.
x and y Returns true if x and y are both true.
x or y Returns true if either x or y or both are true.
x mod y Returns the remainder of x divided by y.

Examples:

The following will remove all songs that have a queue number greater than or equal to 10, or have a length less than 3 minutes. 

remove number >= 10 or length < 180

Remove songs added by 愛美#5352.

remove requester is 愛美#5352

Remove songs added by 愛美#5352 or you.

remove requester is 愛美#5352 or requester is me

Remove all even-numbered songs in the queue.

remove number is even
