ジュックのコマンド

知識を表示

知識を表示します.

help

プレイソング (YouTube) (ギルドだけ)

YouTubeの動画かプレイリストをキューに加えます。

play <youtube_url | string>

ボイスチャネルに入る (ギルドだけ)

自分のボイスチャネルに入ります。

join

スキップする (ギルドだけ)

プレイしている曲をスキップし次のてキューの曲をプレイします。

skip

曲を削除する (ギルドだけ)

曲をキューから削除します。

remove <songs | variadic (<queue_position | number>)>

音量を変更する (ギルドだけ)

音量を変更します。音量は０－１００以内はずです。

volume <volume | number>

キューを表示する (ギルドだけ)

キューの曲を表示します。

queue

プレイしている曲を表示する (ギルドだけ)

プレイしている曲の知識を表示します.

now

YouTubeを引く (ギルドだけ)

曲のためにYouTubeを引きます.

search <query | variadic (<search_term | string>)>

シャフル (ギルドだけ)

今のプレイリストをシャフルする。

shuffle

ボイスチャネルに出る (ギルドだけ)

今のボイスチャットに出てキューを削除します。

leave

知識を表示する

今のページをリンクします。

info

アドバンスコマンド

削除する (アドバンス) (ギルドだけ)

欲しいかたのためにアドバンス削除コマンドがあります。

remove <remove <filters | variadic (<filter_token | string>)>

以下のバリアブルはフィルターに使うことができる。

  • title 曲のタイトル
  • requester_id 曲を加えた方のID
  • requester 曲を加えた方（メンションはずです)
  • length 秒で曲の期間
  • number キューの曲の数
  • me あなた

以下のオペレーターはフィルターに使うことができる。

Operator Description
x contains y Returns true if x contains y (case-insensitive).
x is y Returns true if x equals y.
x is not y Returns true if x does not equal y.
x > y Returns true if x > y.
x >= y Returns true if x >= y.
x < y Returns true if x < y.
x <= y Returns true if x <= y.
x and y Returns true if x and y are both true.
x or y Returns true if either x or y or both are true.
x mod y Returns the remainder of x divided by y.

例え:

キューの番目が１０以上か３分以下なら、曲を削除します。 

remove number >= 10 or length < 180

愛美#5352が加える曲を削除します。

remove requester is 愛美#5352

愛美#5352が加えるか自分が加えるなら、曲を削除します。

remove requester is 愛美#5352 or requester is me

キューの番目は偶数なら、曲を削除します。

remove number is even
